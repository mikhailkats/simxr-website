# Nebius.com Design Reference

## Key characteristics:
- White/near-white background (#FFFFFF / #F9F9F9)
- Very dark near-black text (#0A0A0A / #111)
- Accent: bold lime/yellow-green gradient blob on hero — but for SIM XR we use blue accent from pitch deck (#0057FF)
- Nav: white bg, thin bottom border, logo left, links center, CTA buttons right (dark filled + outlined)
- Hero: large bold left-aligned headline (700-800 weight), short subtitle, 2 CTA buttons, abstract graphic/shape right
- Sections: generous whitespace, 80-120px vertical padding
- Feature grid: 3-col, no card borders, just icon + bold title + short text
- Typography: very large display (56-72px), clean sans-serif, tight letter-spacing
- NO decorative elements, no gradients on text sections
- Dividers: very subtle gray lines or just whitespace
- CTA buttons: dark filled (#111 bg, white text) + ghost/outlined
- Footer: light gray bg, simple grid
